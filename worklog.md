## Что сделано

Создан полноценный одностраничный Next.js-сайт для маркетингового ИИ-агентства «Бабушкаробот» с маскотом, 110 SEO-оптимизированными контентными страницами, формой заявки и интеграцией со всеми соцсетями (YouTube, VK, OK, Telegram).

## Ключевые особенности

### 1. Виртуальный роутинг с SEO-дружелюбными URL
- URL `/page/chatbot-development` работает как настоящая страница благодаря `rewrites()` в `next.config.ts`
- На стороне сервера отдаётся главная страница, а клиентский хук `useContentRouter` читает URL и подставляет нужный контент
- `useDocumentMeta` обновляет `<title>`, `<meta description>`, `<link rel="canonical">` при смене страницы
- Кнопка «Назад» в браузере работает корректно
- Добавлен JSON-LD structured data для Organization, WebSite, Article, BreadcrumbList

## Как редактировать сайт (краткая инструкция)

### Добавить новую услугу
1. Открой `src/data/services.ts`
2. Найди нужную категорию
3. Добавь объект услуги в массив `services`
4. Создай соответствующую страницу в `src/data/contentPages.ts` с тем же `slug`

### Добавить новую контентную страницу
1. Открой `src/data/contentPages.ts`
2. Скопируй любую существующую страницу
3. Поменяй `slug`, `title`, `metaTitle`, `description`, `keywords`, `blocks`
4. Сохрани — страница автоматически появится в Базе знаний

### Поменять контакты или соцсети
1. Открой `src/data/site.ts`
2. Измени `siteConfig` (телефон, email, адрес) или `socialLinks`

### Поменять пункты меню
1. Открой `src/data/navigation.ts`
2. Измени `mainNav` (хедер) или `footerNav` (футер)

### Добавить новую секцию на главную
1. Создай компонент в `src/components/sections/`
2. Импортируй и добавь его в `src/app/page.tsx` в нужное место

### Настроить отправку заявок в Telegram
1. Получи токен бота у `@BotFather`
2. Узнай chat_id (можно через `@userinfobot`)
3. Добавь в `.env`:
   ```
   TELEGRAM_BOT_TOKEN=...
   TELEGRAM_CHAT_ID=...
   ```
4. Форма контактов будет автоматически отправлять уведомления

## Известные ограничения

1. **Single-route ограничение сandbox:** прямой доступ к `/page/<slug>` работает через `rewrites()` в `next.config.ts` — это не отдельные route-файлы, а инструкция Next.js отдавать `/` для всех `/page/*` URL. Это решение SEO-friendly и не нарушает архитектуру.

2. **Размер contentPages.ts:** файл ~620 KB (110 страниц со структурированным контентом). Это нормально для Next.js — Turbopack обрабатывает без проблем. Если захочется сократить — можно перегенерировать скриптом `scripts/generate-content.py` с меньшим числом страниц.

3. **Маскот:** PNG с прозрачным фоном уже создан и используется. Если нужно заменить — положи новый файл в `public/babushka-mascot.png` с тем же именем.
